*HTML*
Linguagem de marcação para criar estruturas web, exemplo: textos
imagens, tabelas, formulários, links, etc...

*CSS*
Linguagem de estilos, podemos alterar características da
linguagem HTML, como por exemplo: fontes, cores, bordas...

*JavaScript*
É uma linguagem de programação. Nessa etapa o foco do JavaScript será trabalhar com interações como: ao digitar, ao clicar, passar o cursor sobre algum elemento...
